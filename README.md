# dot files

repo to keep dot files and command line tools

motivated by [invisibleroads](https://github.com/invisibleroads/scripts)


```bash
sudo dnf install stow
# ...
cd $HOME
git clone git@github.com:salah93/dotfiles
cd dotfiles
stow scripts
stow git
stow fish
stow vim
stow tmux
stow alacritty
```
